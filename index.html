Estrutura HTML

!DOCTYPE html
Define o tipo de documento como HTML5.
head
É o cabelhaçalho do documento, e contém metadados, título da página, link.
body
É o corpo do documento.
footer
É o pé do documento, serve para finalizar e/ou colocar informações importantes no final da página.
HTML
HTML é um acrônimo para Hypertext (hipertexto) Markup (marcação) Language (linguagem), ela não é uma linguagem de programação em si, porém é uma linguagem na forma de escrever, tendo sintaxe e semântica, Hypertext é uma marcação especifica que nos leva a outro texto, isso mais no passado, agora sendo relacionado a imagens.
